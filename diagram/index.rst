用图表来把握项目架构和业务逻辑
=============================

思维导图 + 两UML图 + 两数据库图 = 5种图

软件项目的环节
---------------

- 需求分析、概要设计、详细设计、开发

思维导图
---------
描述各级主题之间的隶属关系

泳道图（流程图）
----------------
一种UML活动图。本质上是一种跨职能流程图

序列图
--------
一种UML交互图。描述了各对象在其生命线上按照约定顺序发送消息的交互行为。由对象、生命线、触发事件、消息组成

类图
-------
一种UML结构图。

数据库关系图（E-R图）
--------------------
即实体-关系图

数据库模型图
-------------
描述了数据库表的字段以及表之间的关系
